# taskratchet-web

- [Install Elm](https://guide.elm-lang.org/install.html)
- `npm install -g elm-format`
- `elm make src/App.elm --output=app.js`
- Open `index.html` in a browser

## Running the Tests

- `npm install -g elm-test`
- `elm-test`

## Other Commands

- `elm repl` - cli interpreter
