# taskratchet-web

Live at <https://app.taskratchet.com/>.

## Development

```bash
nvm use
pnpm install
pnpm start
open http://localhost:3000/
```

## Production

The app is automatically deployed to production when changes are merged into the `master` branch.

```bash
pnpm build
```

## Tests

```bash
pnpm test
```
